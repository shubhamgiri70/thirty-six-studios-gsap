# thirty-six-studios-gsap

Thirtysixstudios Digital Experience
This is a digital experience website designed for Thirtysixstudio, showcasing immersive digital design, motion, and creative technology. It includes interactive elements such as canvases displaying images with dynamic transitions and animations.

Features
Interactive Canvas: On click, an animated circle appears, followed by a change in background color, and displays images from a series of canvases.
Locomotive Scroll Integration: Adds smooth scrolling effects for a modern, visually appealing experience.
GSAP Animations: Provides various animation effects such as fading in canvases, growing animations, and dynamic color changes.
Responsive Design: The website is responsive and adapts to different screen sizes and devices.
Tech Stack
React.js: The framework used for building the interactive components.
GSAP: For smooth animations and transitions.
Locomotive Scroll: For smooth scroll effects.
Tailwind CSS: For rapid, utility-first styling of components.
